データ収集（随時〜毎週）

1 レシート撮影・アップロード 手動 随時

看護師がGoogleフォームからレシート画像をアップロード（最大10枚）。GASが自動で「経費_未処理」フォルダに移動。

2 OCR自動解析 AI 毎週金曜

OpenAI Vision APIでレシートを自動解析。日付・金額・店名・品目・経費区分を自動判別。

npm run expense:sync
記録・集計（毎週〜月末）

3 Google Sheets転記 自動 毎週金曜

解析結果をGoogle Sheetsに自動記録。看護師名・費目・金額・日付・備考・領収書URLを記入。

4 看護師別集計 自動 月末

SUMIF関数で各看護師の合計金額を自動計算。月末に実行。

npm run expense:summary
出力・反映（月末〜月初）

5 月次レポートPDF生成 自動 月末

経費区分別にグループ化したPDFレポートを生成。税理士への提出用。

npm run expense:report

6 給与反映・確認 手動 月初

月初にGoogle Sheetsで前月の経費を確認し、給与に反映。

ふくろく訪問看護ステーション ／ 最終更新: 2026年4月3日

1. レシート撮影・アップロード 随時

アップロード方法

  • Googleフォームにアクセス（スマホからOK）
  • 担当者名を選択
  • レシート画像を添付（最大10枚）
  • 送信 → GASが自動で「経費_未処理」フォルダに移動

撮影のコツ

  • 明るい場所で、レシート全体が入るように
  • 文字がはっきり見える（ピント合わせ）
  • 影が入らないよう真上から撮影
  • 折れ・しわを伸ばしてから撮影

2. OCR自動解析 毎週金曜

コマンド（3つの方法）

実行例
npm run expense:sync    # フォーム回答を一括処理（推奨）
npm run expense:batch   # 未処理フォルダを一括処理
npm run expense:ocr <画像> <名前>  # 1枚ずつ処理

自動判別される経費区分（10種類）

消耗品費 文房具・事務用品
医療材料費 ガーゼ・グローブ
旅費交通費 電車・バス・タクシー
車両費 ガソリン・駐車場
接待交際費 会食・手土産
会議費 カフェ打ち合わせ
研修費 書籍・セミナー
通信費 切手・宅配便
福利厚生費 健康診断
雑費 その他
API料金：レシート1枚あたり約1.5円。月100枚で約150円。

3. Google Sheets転記 毎週金曜

シートの列構成

  • A列：看護師名
  • B列：費目（経費区分）
  • C列：金額
  • D列：日付
  • E列：備考（店名・品目）
  • F列：領収書URL（Google Driveリンク）

手動入力も可能

実行例
npm run expense "富田祐実" "健康診断" 10725
npm run expense "福嶋千亜季" "駐車場代" 5000

4. 看護師別集計 月末

出力イメージ

G列〜H列に自動追加
看護師名        合計金額
─────────────────────
富田祐実        15,240円
福嶋千亜季       8,900円
志摩孝政        12,500円
SUMIF関数で自動計算されるので、新しい経費を追加するたびに自動更新されます。

5. 月次レポートPDF 月末

コマンド

実行例
npm run expense:report           # 当月分
npm run expense:report 2026-02   # 指定月

PDF内容

  • タイトル：「ふくろく訪問看護ステーション 経費精算報告書」
  • 対象期間・件数・合計金額
  • 経費区分ごとにテーブル表示
  • 出力先：output/expense_report_YYYYMM.pdf

6. 給与反映・確認 月初

月初のチェック項目

  • Google Sheetsで前月分の経費を確認
  • 看護師別集計の金額を確認
  • 給与計算に経費精算分を反映
  • PDFレポートを税理士に提出

推奨スケジュール

  • 毎週金曜：npm run expense:sync（未処理を一括処理）
  • 月末：npm run expense:summary（集計）
  • 月末：npm run expense:report（PDF生成）
  • 月初：確認 → 給与反映
